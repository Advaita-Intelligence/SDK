# Acai Proxy Server

Example Express server acts as a proxy for Acai analytics, allowing you to intercept, log, and forward Acai tracking requests.

## Features

- ✅ Proxy single Acai events
- ✅ Proxy batch Acai events  
- ✅ CORS support for browser requests
- ✅ Request logging and debugging
- ✅ Health check endpoint
- ✅ Test page for validation
- ✅ Request monitoring and debugging

## Quick Start

### Option 1: Run from Root Directory

```bash
# From the project root directory
npm run proxy        # Start the proxy server
npm run proxy:dev    # Start with auto-restart (development)
```
